Features
  • Automatic roles are used to automatically grant and revoke privileges.
    • Grant SSH access to hosts in a layer (with and without sudo)
  • Policies can be used to scope records by name and ownership.
    • Declare a scoped policy
  • Public keys can be associated with user records.
    • Public keys can be managed through policy.
    • The latest public key version is the one provided by the pubkeys API.
  • Users and groups can be added to groups
    • Create a group of groups
    • Add an administrator to a group
  • YAML anchors can be used for bulk actions
    • Group records for bulk actions
